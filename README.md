# Система бронирования гостиничных номеров

Веб-приложение для бронирования номеров (Client-Server архитектура).
Стек: **Python (FastAPI)**, **SQLite**, **JavaScript**, **HTML/CSS**.

## Как запустить

1.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

2.  Инициализируйте базу данных (создаст админа и тестовые номера):
    ```bash
    python seed_data.py
    ```

3.  Запустите сервер:
    ```bash
    python -m uvicorn main:app --reload
    ```

4.  Откройте в браузере:
    *   Клиент: `http://127.0.0.1:8000`
    *   Администратор: `http://127.0.0.1:8000/admin`

## Тестовые данные
*   **Администратор:** Логин: `admin`, Пароль: `admin`
*   **Клиент:** Можно зарегистрировать любого пользователя через форму.

## Структура проекта
*   `main.py` — логика сервера (API).
*   `models.py` — модели базы данных (SQLAlchemy).
*   `seed_data.py` — скрипт для заполнения БД начальными данными.
*   `static/` — фронтенд (HTML, CSS, JS, Картинки).
